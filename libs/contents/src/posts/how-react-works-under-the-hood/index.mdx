# شروع کار با جاوا اسکریپت

JavaScript is a versatile and powerful programming language used for web development. It allows you to add interactivity and dynamic behavior to your websites. In this article, we'll cover some basics to get you started with JavaScript.

## Writing JavaScript

You can include JavaScript code in your HTML documents using `<script>` tags. Here's a simple example:

```jsx
<script>// Your JavaScript code goes here alert("Hello, World!");</script>
```
